# 📊 WEB-SYSTEM-REVERIE – Admin Dashboard & Client Management

> A full-stack web application designed to streamline administrative operations through client management, order processing, dashboard analytics, and business reporting.

---

## 📌 Project Overview

**WEB-SYSTEM-REVERIE** is a comprehensive administrative and client management system developed as a collaborative academic project. The application provides administrators with centralized tools to manage clients, process orders, monitor business activities, and generate reports for informed decision-making.

The project focuses on improving operational efficiency through an intuitive dashboard, organized data management, and responsive user interfaces.

## 🚀 Installation

### Prerequisites

- PHP
- MySQL
- XAMPP

### Clone the Repository

```bash
git clone https://github.com/RechelleBorbe67/WEB-SYSTEM-REVERIE.git
```

### Setup

1. Place the project inside the `htdocs` folder.
2. Import the provided MySQL database.
3. Start Apache and MySQL using XAMPP.
4. Open your browser and navigate to:

```
http://localhost/WEB-SYSTEM-REVERIE
```
